Elementos principales de Angular

Las propiedades, metodos/funciones de su clase (@Component), se pueden utilizar en el HTML -> Interpolación

Interpolación

La interpolación se usa entre dobles llaves "{{ } }"

Las propiedades de la clase se pueden utilizar en el HTML

Propiedad nombre -> {{ nombre }}

Las propiedades o funciones/metodos pueden utilizar this (opcional)

Propiedad apellido -> {{ this.apellido }}

Los metodos tambien pueden llamarse desde la interpolación, son usados para mostrar datos calculados

Para los metodos es necesario agregar sus parentesis () -> {{ metodo2() }}, de lo contrario no se va a ejecutar {{ metodo2 }}

Directiva *ngIf

La directiva *ngIf es una directiva estructural, sirve para mostrar u ocultar el elemento donde este la directiva

ngIf necesita de una evaluación booleana -> true o false

Las evaluaciones pueden ser de propiedades directas, o de operaciones

Directiva *ngFor

La directiva *ngFor usa un arreglo o un objeto iterable para recorrer el numero de elementos que contenga el arreglo o el objeto iterable

Arreglo -> [] un arreglo puede contener cualquier cosa

[1,2,3,4,5] numeros, letras y textos ['abc', 'b', 'c'], o combinado [1, "uno", "cualquier tipo"]

Ejemplo: [1,11,34,21]

El tamaño de un arreglo viene dado por el numero de elementos que los contenga, en este caso: Tamaño = 4

Los indices inician en 0, por lo tanto

El primer elemento del arreglo es: arreglo[0] = 1

Y el último elemento del arreglo es: arreglo[3] = 21

Volviendo al *ngFor lo que hace es recorrer cada uno de los elementos de un arreglo o de un objeto iterable

Iterar arregloEjemplo

Para utilizar un *ngFor es necesario definir una variable temporal. La variable temporal guarda el item del recorrido actual

Se puede llamar como prefieran.

Y por ultimo se necesita el arreglo u objeto que se quiera recorrer.

  • Valor: {{ miVarTemporal }}

Podemos obtener de metodos ya existentes de nuestro arreglo, ciertas cosas, como su tamaño.

Tamaño: {{ arregloEjemplo.length }}

Se puede accesar a los items de un arreglo con su indice

arregloEjemplo[indice (numero)] = {{arregloEjemplo[0]}}

Si quisieran accesar al ultimo elemento del arreglo tendrian que hacerlo de la siguiente manera

arregloEjemplo[tamaño del arreglo - 1] = {{ arregloEjemplo[arregloEjemplo.length - 1] }}

ngModel

ngModel es una directiva que sirve para enlazar los datos de un input con una propiedad de la clase del componente

Para utilizar ngModel es necesario importar el Modulo FormsModule, este se pone en el modulo donde el componente este registrado, en este caso (AppModule)

Para "enlazar" un input a una propiedad, es necesario tenerla en el ts (componente)

El ngModel se tiene que poner entre [()]

El que tenga [] y () significa que es doble binding

binding = enlazar

Todo cambio es automatico en un binding -> {{propiedadInput}}

Los inputs pueden tener formato, como números, tipo password, etc.

El tipo se cambia en el atributo type

Componentes

Los componentes pueden ser de muy pequeños como un texto, un botón, hasta una página web completa.

Los componentes se pueden reutilizar.

Dependiendo de la versión de Angular, los componentes pueden estar "sin modulo" o pertenecer a uno.

Para cada componente es necesario crear su clase class y su decorador @Component

En los componentes podemos trabajar el HTML, CSS Y TS en un mismo archivo, o separarlos en diferentes archivos.

Ejemplo (Un archivo -> carpeta individual)

Para mostrar un componete, se hace uso de su selector:

Hay dos formas, abrir y cerrar (/)

La otra manera es una etiqueta void
